# ❤️ Heart Disease Prediction Web App

## 📌 Project Overview
This project is a **Machine Learning-based web application** that predicts the risk of heart disease based on user medical inputs. The application is built using **Flask (Python)** and provides real-time predictions through a simple and interactive user interface.

-----

## 🚀 Features
- 🧠 Machine Learning model for prediction
- 🌐 Web interface using Flask
- ⚡ Real-time prediction (High Risk / Low Risk)
- 🎨 User-friendly UI with CSS styling
- 📊 Uses medical parameters for accurate results

-----

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS
- **Backend:** Flask (Python)
- **Machine Learning:** Scikit-learn
- **Libraries:** Pandas, NumPy
- **Dataset:** Cleveland Heart Disease Dataset

## 🧪 Input Parameters
The model takes the following inputs:

- Age
- Sex (1 = Male, 0 = Female)
- Chest Pain Type (0–3)
- Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope
- Number of Major Vessels (CA)
- Thalassemia
## ⚙️ How It Works
1. User enters medical details in the web form
2. Data is sent to Flask backend
3. Data is preprocessed and scaled
4. Machine Learning model makes prediction
5. Result is displayed on UI
